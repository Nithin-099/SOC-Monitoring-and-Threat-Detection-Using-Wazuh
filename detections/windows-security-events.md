# Windows Security Event Monitoring

## Objective

Monitor Windows security-related activities through centralized log collection.

## Events Monitored

* User logon events
* User logoff events
* Account creation events
* Security configuration changes
* System events

## Detection Method

Wazuh Agent collected Windows Event Logs and forwarded them to the Wazuh Manager for analysis.

## Result

Security events were successfully centralized and visualized through the Wazuh Dashboard.

## Benefits

* Improved endpoint visibility
* Centralized monitoring
* Faster incident investigation
