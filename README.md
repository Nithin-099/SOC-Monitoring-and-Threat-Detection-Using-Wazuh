# SOC-Monitoring-and-Threat-Detection-Using-Wazuh

## Project Overview

This project demonstrates the deployment and configuration of a Security Information and Event Management (SIEM) solution using Wazuh. The objective was to build a centralized security monitoring lab capable of collecting logs, detecting security events, performing threat hunting, and assessing endpoint compliance.

The lab consists of a Wazuh Server running on Ubuntu 22.04 and a Windows 11 endpoint monitored through the Wazuh Agent.

---

## Lab Architecture

```text
Windows 11 Endpoint
        │
        ▼
   Wazuh Agent
        │
        ▼
   Wazuh Manager
        │
        ▼
    OpenSearch
        │
        ▼
  Wazuh Dashboard
```

## Implementation

### Wazuh Server Setup

* Installed Wazuh Manager, Dashboard, and OpenSearch on Ubuntu.
* Configured API communication and dashboard integration.
* Verified service health and log collection.

### Endpoint Monitoring

* Installed Wazuh Agent on Windows 11.
* Registered the endpoint with the Wazuh Manager.
* Verified agent connectivity and event forwarding.

### Threat Hunting

* Monitored Windows security events.
* Investigated user account activities.
* Analyzed logon and authentication events.
* Reviewed security alerts generated by Wazuh rules.

### Configuration Assessment

* Performed CIS Microsoft Windows 11 Enterprise Benchmark assessment.
* Identified failed security controls and hardening opportunities.
* Evaluated compliance score and remediation recommendations.

---

## Security Use Cases

### User Account Creation Detection

A new Windows user account was created using:

```cmd
net user TestUser Password@123 /add
```

Wazuh successfully detected and logged the event, generating security alerts for investigation.

### Windows Log Monitoring

Collected and analyzed:

* Successful logons
* User account modifications
* System activity events
* Security-related Windows events

### Compliance Monitoring

Performed CIS benchmark checks against the Windows endpoint to identify misconfigurations and security weaknesses.
